# FactorLab — 因子分层回测看板(单因子)

A 股市值因子的分层回测可交互看板。支持调节分组数、时间区间,实时查看各组累计净值、分年度收益、统计指标,并附 AI 生成的因子分析备忘录。

> 个人学习项目。这是 FactorLab 系列的单因子基础版;多因子完整版见 [factorlab-multi](https://github.com/kaiying-coder/factorlab-multi)。

**在线 Demo**:https://factorlab-kxh6tywidu6dq8an7tkbga.streamlit.app

---

## 功能

- **分层回测**:每日按市值因子将股票分为 N 组(可调),组内等权,计算各组次日收益。
- **可视化**:各组累计净值曲线(对数坐标)、分年度收益热力图、全期统计指标(年化收益/波动/Sharpe/最大回撤)。
- **多空组合**:小市值组多头 − 大市值组空头的组合指标。
- **交互**:侧边栏可调分组数与时间区间,图表实时更新。
- **AI 备忘录**:基于回测结果生成的因子分析备忘录(只读展示)。

## 方法要点

- **防前视偏差**:因子用过去信息,收益用次日对齐(`shift(-1)`),避免用到未来数据。
- **分组**:按因子横截面排名均分,排名用 `first` 方法打破并列。
- **架构**:计算与展示分离 + `@st.cache_data` 缓存,响应快。

## 技术栈

Python · pandas · numpy · Streamlit · Plotly · matplotlib

## 本地运行

```bash
pip install -r requirements.txt
streamlit run app_deploy.py
```

## 说明与局限

- **在线看板使用抽样数据**(近 6 个月、部分股票)以满足部署体积限制,因此线上展示的各组收益单调性不明显——这是小样本统计噪声所致,非方法问题。使用全样本(2006–2025 全市场)可清晰看到市值因子的小盘溢价单调效应,详见多因子版仓库。
- AI 备忘录为基于回测结果的示例,线上为只读展示。
- 本项目用于方法学习与展示,**不构成任何投资建议**。
